# requirements-no-tts.txt — Cross without TTS/audio packages

# Use this file if you do not need st-speak, st-voice, or --mp3 rendering.
# TTS packages (soundfile, yakyak, websockets) have wheels for Python 3.10–3.13
# on macOS ARM — there is no Python version reason to avoid them.  Use this
# file if you simply do not want audio support, or need a lighter install.
#
# PYTHON VERSION: 3.10, 3.11, 3.12, or 3.13
#
#   MINIMUM 3.10 — still required (numpy 2.x, scipy 1.15.x, match statements)
#
# Commands unavailable without TTS packages:
#   st-speak, st-voice, st-prep --mp3 / --all
# All other commands (st-gen, st-cross, st-stones, st-fact, st-post …) work normally.
#
# Install:
#   python3.11 -m venv .venv && source .venv/bin/activate
#   pip install -r requirements-no-tts.txt
#   bash script/symbolic_links.bash
#
# For full TTS support use requirements.txt with Python 3.10–3.13.
